# System Info Pro

**Versão 6.0**  
Monitoramento completo e moderno do seu computador em tempo real.

Uma ferramenta leve, bonita e profissional para visualizar todas as informações do seu sistema de forma organizada.

---

### ✨ O que o System Info Pro oferece?

- Dashboard com gráficos em tempo real (CPU e RAM)
- Informações detalhadas de CPU, Memória, Discos, Rede e GPU
- Monitoramento de temperatura e sensores
- Lista de processos em execução (com opção de finalizar)
- Benchmark simples de CPU
- Leitura SMART dos discos (saúde do SSD/HDD)
- Informações da placa-mãe e BIOS
- Interface moderna e fácil de usar

---

### 📥 Como usar

1. Baixe o arquivo `SystemInfoPro.exe`
2. Execute o programa (recomendado rodar como **Administrador** para melhores resultados)
3. Pronto! Todas as informações do seu sistema aparecerão organizadas.

> **Dica:** Clique nas abas laterais para navegar entre as seções.

---

### 📝 Observações Importantes

- Para melhores resultados, **execute o programa como Administrador**.
- A aba **SMART** requer que o `smartmontools` esteja instalado no sistema.
- Algumas informações avançadas (como temperatura e GPU) funcionam melhor em Windows.
